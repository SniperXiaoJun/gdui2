## 属性配置
### Row props:
| 参数 | 说明    | 类型 | 可选值  | 默认值   |
|---------- |-------- |---------- |-------------  |-------- |
| gutter| col之间的间距   | number  |    —    |  30  |

### Col props:
| 参数 | 说明    | 类型 | 可选值  | 默认值   |
|---------- |-------- |---------- |-------------  |-------- |
| span| col所占格数(总数12)   | number  |    —    |  12  |
