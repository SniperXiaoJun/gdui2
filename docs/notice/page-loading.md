## 属性配置
### PageLoading.show方法参数
| 参数 | 说明    | 类型 | 可选值  | 默认值   |
|---------- |-------- |---------- |-------------  |-------- |
| loadingText    | 加载中的文字	 | string    |  -  |   'loading'  |
| withMask  | 是否带遮罩层  |  boolean  |  -  |  true  |
